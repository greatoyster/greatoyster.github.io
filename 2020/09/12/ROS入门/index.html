ROS的图概念

  • Nodes: A node is an executable that uses ROS to communicate with other nodes. 和其他节点通信的可执行文件
  • Messages: ROS data type used when subscribing or publishing to a topic.用在某个话题下发布和订阅的数据类型
  • Topics: Nodes can publish messages to a topic as well as subscribe to a topic to receive messages. 节点可以把数据发送到话题下面，节点也可以订阅的的话题下的数据
  • Master: Name service for ROS (i.e. helps nodes find each other)，相当于是一个管理者，帮助Node之间互相通信
  • rosout: ROS equivalent of stdout/stderr. ROS里面的等价标准输入
  • roscore: Master + rosout + parameter server (parameter server will be introduced later). ROS里面的核心部件，基本得开了roscore了以后，节点，服务这些东西才可以正常运行

Node client libraries

节点利用一个客户端的库来实现它的功能

  • rospy = python client library
  • roscpp = c++ client library
